# FitTrack Diagrams

`ARCHITECTURE.puml` is the source of truth for FitTrack architecture diagrams.
It contains named PlantUML blocks so one command can export every diagram.

## PlantUML Export

From the repository root:

```sh
plantuml -tsvg -o ../architecture docs/diagrams/ARCHITECTURE.puml
plantuml -tpng -o ../architecture docs/diagrams/ARCHITECTURE.puml
```

PlantUML writes generated files under `docs/architecture/`, alongside the tracked PNG exports.

## Astah Handoff

Astah project files are binary project files generated by Astah. Keep the live
project file at `docs/diagrams/astah/fittrack-architecture.asta` after creating
or updating it in Astah, then export PNG/SVG files into
`docs/architecture/`.

Use `ARCHITECTURE.puml` as the import/update source when the Astah PlantUML
workflow is available in the local Astah installation.
